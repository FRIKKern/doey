## Risks & Mitigations

| Risk | Mitigation |
|------|------------|
| P95 latency exceeds 2s target | Hard `--max-time 2.5`; silent fallthrough. No hang possible. |
| Haiku hallucinates a command not in schema | System prompt forbids; schema is input context; `auto_correct` result is still validated against `doey-ctl --help` before exec |
| Destructive command auto-corrected | Static blocklist + mandatory `[y/N]` prompt; blocklist covers `uninstall`, `kill-*`, `purge`, `push --force`, `reset --hard` |
| User has no API key | Silent disable; no prompt, no nag; `doey doctor` shows status |
| Agent pane breaks because of new stdin/stdout flow | Auto-correct prints to stderr, executes via `exec`, preserving the caller's stdin/stdout |
| API key leaked in logs | Redact known secret-bearing flags; never include request body in log line |
| flag.ExitOnError bypass | Documented gap, v2 follow-up task pre-created |
