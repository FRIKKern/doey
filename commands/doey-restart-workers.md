# Skill: doey-restart-workers
## Usage
`/doey-restart-workers` (DEPRECATED)
## Prompt
This command is deprecated. Use `/doey-restart-window` instead. Note: `/doey-restart-window` restarts workers only (not the Watchdog). To restart the Watchdog, kill its process in the Dashboard pane and relaunch with `claude --dangerously-skip-permissions --model haiku --effort low --agent doey-watchdog`.
