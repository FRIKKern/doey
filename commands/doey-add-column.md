# Skill: doey-add-column

Add a worker column to the dynamic grid.

## Usage
`/doey-add-column`

## Prompt
Add a new column of workers to the dynamic grid.

### Step 1: Add column

```bash
doey add
```

### Step 2: Report

Report the result — how many new workers were added and the new grid dimensions. If it failed (e.g., already at max columns), explain why.
