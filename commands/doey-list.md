# Skill: doey-list

List all registered Doey projects and their status.

## Usage
`/doey-list`

## Prompt
Show all registered Doey projects and their session status.

### Step 1: List projects

```bash
doey list
```

### Step 2: Present results

Format the output nicely for the user. Highlight:
- Which projects have active sessions
- Which are stopped
- Session names and project directories
